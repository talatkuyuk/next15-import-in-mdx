### 1. Import A Server Component with `.mjs` extension

The **relative link** is relative to the module which **`MDXRemote`** called,
since the `baseUrl` is setted as `import.meta.url` in the options.

`import Bar from "../../../mdxComponents/Bar.mjs";`

import Bar from "../../../mdxComponents/Bar.mjs";

<Bar />

### 2. Import A Server Component with `.mjs` extension via dynamic import

The **relative link** is relative to the module which **`MDXRemote`** called,
since the `baseUrl` is setted as `import.meta.url` in the options.

`export const Bar = (await import("../../../mdxComponents/Bar.mjs")).default;`

export const Bar2 = (await import("../../../mdxComponents/Bar.mjs")).default;

<Bar2 />
