### Import image via relative path (with html syntax)

Relative path resolution is handled by `recma-mdx-change-props`

import imgUrl from "./blog-assets/image.png";

<img src={imgUrl} height={60} width={60} />

### Import image via via relative path (with markdown syntax)

using 'recma-mdx-import-media', it is possible below syntax

![alt](./blog-assets/image.png)

<img src={"./blog-assets/image.png"} height={60} width={60} />
